# RetailFlow
This project is a full-stack e-commerce web application developed using Ruby on Rails. The platform is designed to simulate a modern online retail environment where users can browse products, create accounts, manage shopping carts, and complete purchases through a streamlined checkout process. The system follows the Model-View-Controller (MVC) architectural pattern, leveraging Rails' built-in tools for routing, database management, and authentication to ensure clean code organization and scalability.

The application includes core e-commerce features such as product listings, user authentication, cart management, order tracking, and an administrative interface for product and inventory control. By utilizing Ruby on Rails conventions and ActiveRecord for database interactions, the project emphasizes maintainability, modular design, and efficient development practices. This project demonstrates full-stack development principles, database integration, and real-world web application architecture.

## Features
- User registration and authentication
- Product catalog with categories and detailed descriptions
- Search and filtering of products
- Add to cart and remove from cart functionality
- Shopping cart management with quantity updates
- Secure checkout process
- Order placement and order history tracking
- Inventory management (stock updates after purchase)
- Create, update, and delete products
- Responsive user interface for browsing and shopping
- Form validation and error handling
- Database integration using ActiveRecord ORM
- MVC architecture for organized code structure
